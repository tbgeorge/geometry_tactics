Controls:
	- A mouse is used to navigate menus and make selections on the map using the left mouse button.
	- While holding the right mouse button down, the WASDQE keys can be used to move the camera.
		- While in this mode SHIFT will increase the speed of camera movement
	- Arrow keys can be used to move the camera between pre-defined camera positions
	- The ESCAPE key is used to go back one menu while in play mode.
	- The ESCAPE key is also used to INTERRUPT play when all actors are controlled by AI.

	- F3 can be used to toggle the debug overlay.
	- The '~' key can be used to access the developer console.
